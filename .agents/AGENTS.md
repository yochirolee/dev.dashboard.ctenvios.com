# AGENTS

Project: CTEnvios dashboard (React + TypeScript + Vite).

## Essentials
- Package manager: yarn
- Dev server: `yarn dev`
- Build: `yarn build` (runs `tsc -b` then `vite build`)
- Lint: `yarn lint`
- Preview: `yarn preview`
